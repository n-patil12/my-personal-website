# Somla: Sharing Stories, Cultivate Conversations, and Spread it to the World.

Originating from the Swahili word "Soma" (meaning to "read or study"), Somla is a Vite + React web application that has online books for people 
to be able to read, all day, any day. Each book also comes with a feature where people can have live, virtual conversations, where they can contribute
their thoughts and perceptions of the book, both as a speaker or commentator.
The application is meant to act as a stepping stone for people who want to have their "book club experience," but maybe might not have the same
opportunities to do so in their daily lives. This application changes that.
